# Electricity Consumption Patterns

## Project Title
Plugging into the Future: An Exploration of Electricity Consumption Patterns

## Project Description
This project analyzes electricity consumption patterns using Tableau. Interactive dashboards and stories help visualize electricity usage and identify trends.

## Objectives
- Analyze electricity consumption data.
- Create interactive dashboards.
- Present insights using Tableau Story.
- Publish the dashboard on Tableau Public.

## Dataset
- File Name: Consumption.csv

## Tools Used
- Tableau Desktop
- Tableau Public
- GitHub

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Data Preparation
4. Data Visualization
5. Dashboard Creation
6. Story Creation
7. Publish to Tableau Public
